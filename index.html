How to Install and Configure Percona Server on Amazon Linux

Prerequisites

  • Amazon Linux 2 or 2023 instance
  • Sudo privileges
  • RDS endpoint, username, and password

1️⃣ Update System and Install Dependencies

sudo yum update -y
sudo amazon-linux-extras install epel -y
sudo yum install -y epel-release perl perl-core perl-DBI perl-DBD-MySQL wget tar curl

# Verify Perl modules
perl -MEnglish -e 'print "English module OK\n"'
perl -MDBI -e 'print "DBI OK\n"'
perl -MDBD::mysql -e 'print "DBD::mysql OK\n"'

2️⃣ Install Percona Toolkit

# Download Percona Toolkit 3.7.0
wget https://downloads.percona.com/downloads/percona-toolkit/3.7.0/binary/tarball/percona-toolkit-3.7.0_x86_64.tar.gz

# Extract
tar -xzf percona-toolkit-3.7.0_x86_64.tar.gz
cd percona-toolkit-3.7.0_x86_64/
sudo cp bin/* /usr/local/bin/

# Cleanup
cd ..
rm -rf percona-toolkit-3.7.0_x86_64 percona-toolkit-3.7.0_x86_64.tar.gz

# Verify installation
pt-online-schema-change --version

3️⃣ Install MySQL Client

# Add MySQL 8 repository
wget https://dev.mysql.com/get/mysql80-community-release-el7-8.noarch.rpm
sudo rpm -ivh mysql80-community-release-el7-8.noarch.rpm

# Install MySQL client
sudo yum install -y mysql-community-client

# Cleanup
rm mysql80-community-release-el7-8.noarch.rpm

# Verify installation
mysql --version

4️⃣ Download and Configure RDS SSL Certificate

# Download SSL certificate
wget https://truststore.pki.rds.amazonaws.com/global/global-bundle.pem -O /home/ec2-user/rds-combined-ca-bundle.pem

# Secure the certificate
chmod 600 /home/ec2-user/rds-combined-ca-bundle.pem

# Verify content
cat /home/ec2-user/rds-combined-ca-bundle.pem

5️⃣ Test MySQL Connection to RDS

mysql -h **** -u **** -p \
--ssl-mode=VERIFY_CA \
--ssl-ca=/home/ec2-user/rds-combined-ca-bundle.pem \
-e "SELECT @@hostname, @@version;"

Replace **** with your RDS endpoint and username.

6️⃣ Configure MySQL Client (~/.my.cnf) for Convenience

# Create configuration file
vim ~/.my.cnf

# Add the following:
[client]
host=****
user=****
password=****
ssl=1
ssl-ca=/home/ec2-user/rds-combined-ca-bundle.pem

# Secure configuration file
chmod 600 ~/.my.cnf

7️⃣ Test MySQL Client Using ~/.my.cnf

mysql -p -e "SELECT @@hostname, @@version;"
# The -p flag prompts for the password if not specified in the config

8️⃣ Using Percona Toolkit with ~/.my.cnf

pt-online-schema-change \
--alter "ENGINE=InnoDB" \
D=ypp_uat,t=partner \
--ask-pass \
--alter-foreign-keys-method=auto \
--progress=time,30 \
--chunk-size=1000 \
--statistics \
--dry-run
# Always use --dry-run first. Use --execute only after verifying.
Notes:
  • All dnf commands are removed because Amazon Linux 2 uses yum.
  • Avoid unnecessary packages like MariaDB if only MySQL client is required.
  • Always test Percona Toolkit commands with --dry-run before executing on production.
